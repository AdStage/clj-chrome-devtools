set-auto-attach

(set-auto-attach)(set-auto-attach {:as params, :keys [auto-attach wait-for-debugger-on-start]})(set-auto-attach connection {:as params, :keys [auto-attach wait-for-debugger-on-start]})
Controls whether to automatically attach to new targets which are considered to be related to this one. When turned on, attaches to all existing related targets as well. When turned off, automatically detaches from all currently attached targets.

Parameters map keys:
  :auto-attach          Whether to auto-attach to related targets.
  :wait-for-debugger-on-start Whether to pause new targets when attaching to them. Use <code>Runtime.runIfWaitingForDebugger</code> to run paused targets.
